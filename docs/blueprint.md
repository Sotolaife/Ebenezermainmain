# **App Name**: Pavilion Art Gallery

## Core Features:

- Home Page Showcase: Display a featured artwork with a tagline and artist introduction on the home page.
- About Page with Artist Story: Present the artist's journey and philosophy with elegant typography and a minimalist portrait.
- Interactive Portfolio Gallery: Showcase artworks in a dynamic masonry or grid layout, with zoom and fade-in effects.
- Artwork Detail Modals: Implement modal/lightbox views for each artwork, including titles, descriptions, and a WhatsApp inquiry button.
- Contact Page: Provide a clean and elegant contact form, redirecting inquiries to WhatsApp or mailto.
- Floating WhatsApp Button: Maintain a floating 'Contact via WhatsApp' button across all pages for immediate communication.

## Style Guidelines:

- Primary color: Off-white (#F8F8FF) to provide a clean, gallery-like backdrop that enhances the artworks.
- Background color: Light gray (#E8E8E8) provides a subtle contrast, adding depth without distracting from the art.
- Accent color: Warm gray (#808080) will provide a sophisticated highlight for interactive elements.
- Headings font: 'Poppins' sans-serif font for bold headings
- Body font: 'Lora' serif font, giving a vintage and literary feel for artist story and artwork description
- Emphasize large white spaces to create a sense of calm and high-end gallery aesthetics.
- Apply subtle fade-in animations on scroll and hover zoom on artworks to engage users smoothly.
- Use minimalist icons related to navigation and contact options, keeping the visual language subtle and elegant.
